# Word Puzzle 题库审查

当前 `word puzzle` 实际从 `sentences` 中抽题，而不是使用空的 `app/src/main/assets/json/word_puzzle.json`。下面这些句子不适合继续用于“5词拖拽排序”模式，原因主要分为三类：句子残缺、语义生硬、作为拼图提示不够自然。

| ID | 原句 | 问题 | 修改建议 |
| --- | --- | --- | --- |
| 10 | 我约会、旅行、拾起了一份新的差事 | “拾起了一份差事”搭配生硬，整句像机翻 | 改为 `我约会、旅行，还找到了一份新的工作` |
| 15 | 本州西海岸日本艺术博览会和新观光游 | 名词堆叠，缺谓语，像标题不是句子 | 改为 `本州西海岸举办了日本艺术博览会和新观光游活动` |
| 26 | 我在鸭绿江的北京邮电大学上学 | 地点关系混乱，容易误导 | 改为 `我在北京邮电大学上学` |
| 30 | 冰岛奢华住处遍布的岛屿 | 句子残缺，缺主干 | 改为 `这是一座遍布冰岛奢华住处的岛屿` 或直接替换为完整句 |
| 36 | 我的机场度假就是这样开始的 | “机场度假”表达别扭 | 改为 `我的机场假期就是这样开始的` |
| 37 | 我知道你肯定要当个被宰的游客 | 语气偏负面，也不适合作为学习题目 | 改为 `我知道你肯定不想当一个被坑的游客` 或替换为中性句 |
| 41 | 本月，意大利的里雅斯特的游客 | 明显残句，信息不完整 | 改为 `本月，意大利里雅斯特的游客数量明显增加` |
| 42 | 游客中心将在细致的规范下运行 | “细致的规范下运行”不自然 | 改为 `游客中心将按照严格规范运行` |
| 44 | 罗马斗兽场外的中国游客 | 只是名词短语，不是完整句 | 改为 `罗马斗兽场外聚集了许多中国游客` |
| 46 | 菲律宾便是这样一个地方 | 语义过空，提示价值低 | 改为 `菲律宾就是这样一个适合度假的地方` |
| 48 | 这里是如何度过愉快的空中旅行 | 中文表达不通顺 | 改为 `这里介绍如何度过一次愉快的空中旅行` |

建议后续维护规则：

1. `word puzzle` 只使用完整陈述句，避免标题、短语、残句。
2. 单个词块长度最好不超过 5 个汉字，否则在手机屏幕上容易拥挤。
3. 避免过强翻译腔、地理关系错误和过度负面的表达。
4. 如果句子必须依赖上下文才能理解，不适合做拖拽排序题。
